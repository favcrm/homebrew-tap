# homebrew-tap

Homebrew tap for the [FavCRM CLI](https://github.com/favcrm/cli).

## Install

```bash
brew install favcrm/tap/favcrm
```

## Upgrade

```bash
brew upgrade favcrm
```

`Formula/favcrm.rb` is updated automatically by the
[`cli`](https://github.com/favcrm/cli) release workflow on every `v*` tag —
do not edit it by hand.
